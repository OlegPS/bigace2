Markup language: 
CSS

Description:
A basic CSS editor set with Bold, Italic, Stroke through, Lowercase, Uppercase,
Left/Center/Right/Block Alignment, Text indent, Letter spacing, Line height, 
Left/Top/Right/Bottom Padding, Background image and CSS file import.

Install:
- Donwload the zip file
- Unzip it in your markItUp! sets folder
- Modify your JS link to point at this set.js
- Modify your CSS link to point at this style.css
